HPE/Aruba Networking MIBs Loading Order

The first column, "MIB", is the filename of the MIB that you wish to compile into your MIB browser software (the actual MIB name is shown in parentheses).

The second column, "DEPENDENT MIBs", lists the MIB files that are required for the "MIB" in the first column.

This means that for each "MIB" in the first column, the "DEPENDENT MIBs" listed in the second column must be compiled first before the "MIB" can be compiled.

Prior to loading desired MIB, "DEPENDENT MIBs" list to be loaded bottom up order and compiled.

Note that many MIBs, such as RFC1155-SMI, RFC-1212, and RFC-1215, may be already loaded (pre-compiled) into your MIB browser application.

This MIB loading sequence has been tested and verified to succeed on the Windows NT Network Node Manager MIB compiler.  Results may vary on other MIB browser compilers.
